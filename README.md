# Lab 1 - Grade Calculator and File Organizer

## Project Overview
This project consists of two parts:
1. **Python Grade Generator** - Interactive application to calculate student grades
2. **Bash Shell Organizer** - Script to archive CSV files with timestamping

## Files Structure

```
lab1-username/
├── grade-generator.py # Python grade calculator
├── organizer.sh # Bash file organizer
├── README.md # Project documentation
└── archive/ # Created by organizer.sh (auto-generated)
```


## Part 1: Grade Generator (grade-generator.py)

### Features
- Interactive assignment input with validation
- Support for Formative (FA) and Summative (SA) assignments
- Grade validation (0-100%)
- Weight validation (positive numbers)
- GPA calculation (0-5.0 scale)
- Pass/Fail determination (requires ≥50% in both categories)
- CSV export functionality

### Usage
```
python3 grade-generator.py
```

### Organize all files

```
bash organizer.sh
```
